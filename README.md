## Datasets

- [train-val-data.tsv](https://drive.google.com/drive/folders/1myoeLdsOYz8mSvYEhSdMfUszUJlaJR3u?usp=sharing): It contains all sequence pairs for training and validation.

- [data/train-val-split.json](https://github.com/CMACH508/AlphaDrug/data/train-val-split.json): It contains the index of the training pairs and test pairs in [data/train-val-data.tsv](https://drive.google.com/drive/folders/1myoeLdsOYz8mSvYEhSdMfUszUJlaJR3u?usp=sharing).

- [data/testing-proteins.txt](https://github.com/CMACH508/AlphaDrug/data/testing-proteins.txt): It contains all pdbids of the testing proteins which can be downloaded from [PDBbind website](http://www.pdbbind.org.cn/).
